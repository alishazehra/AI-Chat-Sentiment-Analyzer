
# AI Chat Sentiment Analyzer

A simple web-based application that analyzes user input text and determines the sentiment as Positive, Negative, or Neutral.

## 🚀 Features

- Analyze user messages in real-time
- Detect basic sentiment (Positive 😊, Negative 😞, Neutral 😐)
- Clean and responsive user interface
- Instant result display without page reload

## 🧠 How It Works

This project uses a basic keyword-based approach to simulate sentiment analysis.

- Positive words → Positive sentiment
- Negative words → Negative sentiment
- Otherwise → Neutral sentiment

In real-world applications, this would be powered by Natural Language Processing (NLP) models and AI APIs.
